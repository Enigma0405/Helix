# SOP-STER-014: Sterile Filter Integrity Testing

## Purpose
To define the procedure for performing pre-use and post-use integrity testing of sterilizing grade filters (0.22 μm) used in aseptic processing, ensuring that filters maintain their integrity and do not compromise batch sterility.

## Scope
Applies to all sterilizing grade filters used in Upstream and Downstream Processing at the Boston and Dublin manufacturing sites.

## Responsibilities
- **Manufacturing Operator**: Perform the integrity test (Bubble Point or Forward Flow).
- **Quality Assurance**: Review and approve the integrity test results before batch release.

## Procedure
1. Ensure the filter housing is correctly installed and sterilized.
2. Wet the filter with the specified wetting fluid (e.g., Water for Injection) for at least 5 minutes.
3. Connect the automated filter integrity tester (e.g., Palltronic).
4. Run the pre-programmed test sequence for the specific filter part number.
5. Record the physical printout and attach it to the Electronic Batch Record (EBR).

## Acceptance Criteria
- For a standard 10-inch 0.22 μm filter, the Bubble Point must be ≥ 3.2 bar (46.4 psi).
- Forward flow must be ≤ 15 mL/min at the specified test pressure.

## Deviation Handling
If a filter fails the post-use integrity test:
1. Immediately halt the processing line.
2. Quarantine the affected batch.
3. Initiate a Deviation Report (per SOP-DEV-001) within 24 hours.
4. Perform a root cause analysis to determine if the failure was a false failure (e.g., inadequate wetting) or a true filter rupture.
